Qwen3-TTS Voice Clone Demo

上传一段样音，把它变成一个可复用的声音。

这个 demo 走的是官方云端 voice cloning + speech synthesis API。 页面负责交互，后端做代理，这样不会把 API Key 直接暴露到浏览器里。

Step 1

上传 10-20 秒左右的清晰人声，创建 voice token。

Step 2

输入文本，调用同一个模型生成音频。

Step 3

浏览器里试听、下载，继续迭代样音和文本。

注意

只对你自己的声音，或已获得明确授权的声音执行克隆。不要拿去冒充别人。
