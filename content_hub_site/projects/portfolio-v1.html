作品站 V1

这个版本的目标不是“功能很多”，而是先把个人创作资产收拢到一个可访问、可分享、可持续更新的空间。

版本目标

  • 建立清晰栏目：想法、网页、游戏
  • 保证移动端和桌面端都有良好阅读体验
  • 把“浏览”升级成“可联系、可订阅”

当前技术栈

纯静态实现（HTML + CSS + JS），优势是部署快、维护简单、学习成本低。

下一步路线

  • 升级到 Markdown 内容流，减少手改 HTML
  • 补齐真实作品截图和试玩入口
  • 接入固定域名，完善 SEO 和站点统计
