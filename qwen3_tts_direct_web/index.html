QWEN3-TTS VC

输入文字，直接生成克隆音色语音。

这是单文件网页版。你填 API Key，先上传样音得到 voice token，再输入文本生成音频。页面不会把你的 key 存到本地文件。

只克隆你自己声音，或已获得明确授权的声音。不要用于冒充他人。

通用设置

这部分会同时用于“克隆声音”和“文本合成”。

等待操作。

Step 1: 克隆声音

上传样音后会得到一个可复用的 voice token。

建议样音 10~20 秒、单人、安静环境、无背景音乐。

Step 2: 生成音频

输入文本后调用同一模型生成语音。
